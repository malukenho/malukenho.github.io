{: class="marginalia" }
**CubeWorld** was a<br/>2005 toy by Radica Games:<br/>tiny LCD cubes each<br/>containing a stick-figure<br/>creature that reacted to<br/>movement, connected<br/>to neighbouring cubes.

I do not know if you have heard of CubeWorld. It is a tiny device that displays small
animated characters living inside a cube — they walk, sleep, play, and react when you
interact with them. Think Tamagotchi, but open-world and chain-able across multiple cubes.

I had one as a kid. They have been discontinued for years, and the original hardware is
essentially impossible to find. So I decided to build my own — using an
**ESP32-C3** microcontroller and a small OLED display. Since I ended up spending a few
weekends on this and learned a lot along the way, here is the full write-up.

## What Is CubeWorld?

{: class="marginalia" }
The original Radica cubes<br/>connected magnetically.<br/>Walk your character to<br/>the edge and it would<br/>"jump" into the next cube.<br/>My version is a single<br/>cube, but that is enough.

CubeWorld is a tiny interactive desktop companion living on a small screen. The original
device was a 2005 toy — a plastic cube with a 1-bit LCD display showing a stick-figure
creature that reacted to tilt, taps, and neighbouring cubes.

What makes it interesting to re-implement:

- **Sprite engine** — pixel art characters with multi-frame animations
- **State machine** — creature has moods and transitions between them
- **Input handling** — buttons and/or accelerometer for interaction
- **Tiny display** — 128×64 OLED with 1-bit colour, every pixel matters

<span class="kw">struct</span> <span class="ty">Creature</span> {

function getFrameData(s) {
  var t = s.tick;
  switch (s.state) {
    case 'IDLE':
      // blink every ~3 seconds (150 ticks), for 2 frames
      var inBlink = (t % 150 > 145);
      return inBlink ? F.idle1 : F.idle0;
    case 'WALKING':
      return (Math.floor(t / 6) % 2 === 0) ? F.walk0 : F.walk1;
    case 'SLEEPING':
      return F.sleep;
    case 'HAPPY':
      return F.happy0;
  }
  return F.idle0;
}

function updateCreature() {
  creature.tick++;
  creature.stateTimer++;

  if (creature.state === 'WALKING') {
    creature.x += creature.dir;
    if (creature.x <= 5)   { creature.x = 5;   creature.dir = 1;  }
    if (creature.x >= 108) { creature.x = 108;  creature.dir = -1; }
  }

  if (creature.stateTimer >= creature.stateDur) {
    transitionCreature();
  }
}

function transitionCreature() {
  creature.stateTimer = 0;
  var r = Math.random() * 100;
  if (creature.happiness < 30) {
    creature.state = 'SLEEPING';
    creature.stateDur = 200 + Math.floor(Math.random() * 200);
  } else if (r < 40) {
    creature.state = 'WALKING';
    creature.stateDur = 120 + Math.floor(Math.random() * 160);
  } else if (r < 70) {
    creature.state = 'IDLE';
    creature.stateDur = 100 + Math.floor(Math.random() * 200);
  } else {
    creature.state = 'HAPPY';
    creature.stateDur = 80 + Math.floor(Math.random() * 60);
  }
  updateBadge();
}

  SPRITE_FRAMES.forEach(function(sf, idx) {
    var wrap = document.createElement('div');
    wrap.className = 'sprite-frame' + (idx === selectedSpriteIdx ? ' current' : '');
    wrap.onclick = (function(i) { return function() {
      selectedSpriteIdx = i; renderSpriteViewer();
    }; })(idx);

    var lbl = document.createElement('div');
    lbl.className = 'frame-label';
    lbl.textContent = sf.name;

    var cv = document.createElement('canvas');
    cv.className = 'sprite-canvas';
    cv.width  = 10 * PXSCALE;
    cv.height = 16 * PXSCALE;
    var ctx2 = cv.getContext('2d');

    ctx2.fillStyle = '#060a08';
    ctx2.fillRect(0, 0, cv.width, cv.height);

    ctx2.fillStyle = idx === selectedSpriteIdx ? '#c8fce0' : '#7bcdab';
    for (var row = 0; row < 16; row++) {
      for (var col = 0; col < 10; col++) {
        if (sf.data[row][col]) {
          ctx2.fillRect(col * PXSCALE, row * PXSCALE, PXSCALE - 1, PXSCALE - 1);
        }
      }
    }

    wrap.appendChild(lbl);
    wrap.appendChild(cv);
    container.appendChild(wrap);
  });
}

// ─── Copy buttons ──────────────────────────────────────────────
function copyCode(btn) {
  var pre = btn.closest('.code-wrap').querySelector('pre');
  navigator.clipboard.writeText(pre.innerText).then(function() {
    btn.textContent = 'copied!';
    btn.classList.add('copied');
    setTimeout(function() { btn.textContent = 'copy'; btn.classList.remove('copied'); }, 1600);
  });
}
